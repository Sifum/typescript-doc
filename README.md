This is a Learning Typescript project.
doc/* shows how to scaffold this project with gulp,
and other docs show Typescript.
All stuffs are based on the "http://www.typescriptlang.org/docs/tutorial.html"

doc/*目录下的文件为本人从Typescript官网自己翻译下来的文档，将一些重要知识点提取出来，并快速的进行解析，对于具备基本js水平的朋友可以快速查看和入门。
doc下的gulp.doc为本人构建这个学习项目的完全流程，你通过这个流程的学习，不仅能学到Typescript而且能对使用gulp和各种gulp，node插件构建完整项目应用有个更深入的认识。


学习理念："一斑窥豹不如庖丁解牛"
